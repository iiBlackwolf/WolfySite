    Wolfy's privacy of service

    1. Introduction

    Wolfy is a discord bot that provides various features such as moderation, information, music, search, logs, suggestions, economy, and leveling system.

    This privacy policy outlines how user data is collected, used, and protected by the bot and its developer (WOLF).

    2. Collection of Data

    Wolfy collects data such as user IDs, server IDs, message content, and any other data that is necessary to provide the bot's features and to improve the bot's performance.

    This data is collected through the use of the bot's commands and may include data such as user profiles, server information, and message history.

    3. Use of Data

    User data collected by Wolfy is used to provide the bot's features and to improve the bot's performance.

    User data may be used to personalize the bot's responses, to track user activity, and to improve the bot's functionality.

    User data will not be shared or sold to third parties without the user's consent, except as required by law.

    4. Data Retention

    User data collected by Wolfy will be retained for as long as necessary to provide the bot's features and to improve the bot's performance.

    User data will be deleted upon request by contacting the developer of the bot (WOLF)

    5. Data Security

    Wolfy and its developer take reasonable measures to protect user data from unauthorized access, use, or disclosure.

    These measures include data encryption, secure storage, and access controls.

    However, no data transmission over the internet can be guaranteed to be 100% secure.

    5. Changes to Privacy Policy

    The developer of Wolfy reserves the right to update and change this privacy policy at any time.

    Your continued use of Wolfy after any changes to this privacy policy indicates your acceptance of the new terms.

    Contact Information

    If you have any questions or concerns about these terms of service or the use of Wolfy, please contact the developers of the bot.
